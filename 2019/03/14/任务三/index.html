任务三

发表于 2019-03-14

软云实验室第三次爬虫任务

这次网站爬取的网站是重庆市建设工程招标投标网，这个网站本身存在一些问题，谷歌浏览器好像会把每一项的网站当成弹窗给自动拦截了，直接取消拦截即可，还有一个是600页之后的页数好像是崩溃了的。

首先分析网站


这是网站首页，通过点击下一页，发现其网址没有发生变化

右键-检查 进入程序员模式，这里找到每一项的位置，再找到href属性，发现并不是直接存的网址，而是一个js脚本


检查-network-刷新网页 接收到几个包，点击第一个包，会发现该请求为post请求，下方的Form Data有六个参数

再查看网页源代码，在里面找到了一栏为viewstate的属性，但和post请求的viewstate并不相同

开始思考

Form Data中的六个参数中第一个参数猜测是后端用来识别这是个翻页操作，第三个参数每次都发生改变，其它参数要么为空要么不改变，所以可以用字典构造表单，然后用FormRequest发送请求。至于Viewstate参数，在我的多次尝试之后，发现每次post请求发送出去的viewstate参数都是上一个网页的viewstate值，我的理解是viewstate是一种用了一次就不能再用的钥匙。现在可以实现翻页操作了。
那么翻页是可以了，每个项目的网址怎么获得呢？我不知道url是不是直接在js代码中，但我猜不是，似乎没看到过提取js中url的方法，然后我就猜想可能是js发送一个请求，然后服务器返回数据，js再提取出数据中的url网址。然后我就尝试打开了第一个项目，同时抓取返回的包，包确实是真的有，但返回的却是主页的html，相当于把原来的主页刷新了一遍，我试着查看了一下网页源代码，与原先的网页进行比较，发现在网页源代码的开头处多了一串url，打开之后确实是第一个项目的网页。然后查看返回的这个包的请求表单

这里格式和之前翻页的时候是差不多的，只有第一个参数有区别，然后再去点击其他项目，虽然每次都在改变，但很容易找出其中的规律。

到这里似乎这个爬虫就没什么问题了吧？不！我的噩梦才刚刚开始

写代码

因为要重复很多次刷新主页的操作，所以我最开始想的是递归，但viewstate参数是原来网页的内容，我找不到方法把viewstate传给下一个函数，后来我灵机一动，想到了全局变量，通过每次进入主页后改变全局变量来获得viewstate的值，当时真的觉得自己是个小机灵鬼。但…结果并不是。

最开始的时候我加了几个pdb断点，我发现我的程序无法从第一个函数进入第二个函数，经过我多次尝试之后，发现其实是整个程序的运行顺序都是乱的，毫无规律规律可言，我到这里就崩了，上网搜和问别人也不知道怎么描述这个问题，最后终于学长看懂了我的问题了，告诉我scrapy不能在递归的时候改变全局变量，让我先记住这个就行，里面的原因不好解释清楚，而scrapy中函数之间的传参可以使用meta参数。那我只能想其他办法嘛。（这里我卡了两周…）

meta参数只能单向传输，我想的是只有把viewstate参数回传才有用。

只能再继续找办法，然后我想到既然学长说的是不能递归的时候改变全局变量，那循环的时候改变全局变量或类变量应该没问题吧？然后我就写了下面这个

结果….emmm还是一样的。然后我去找了点资料，我猜这个错误的原因大概是：yield request发送请求是异步的，同时进行了几个函数，并非是一个一个按顺序来进行的，导致全局变量viewstate的值就是没有意义了。

之后我从新写了好几次，但可能是先入为主了，不管怎么写，最后都是原来的这种结构。

然后我的思路一下子就理清了，顿时感觉整个世界都不一样了。之后的程序也很快就写了出来。

from ..items import TaskThreeItem

from scrapy.http import Request

def parse(self, response):
    print("......................这里是parse函数...........................")
    page = 1
    project = 1
    Viewstate = ["0"]
    Viewstate[0] = response.xpath("//*[@id='form1']/input[3]/@value").extract()
    postdata_refresh = {
        "__VIEWSTATEGENERATOR":"77E10603",
        "__EVENTARGUMENT":"",
        "__EVENTTARGET":'DataGrid1:_ctl3:_ctl0',
        '__VIEWSTATE':Viewstate[0],
        "FNam":"",
        "Pager1:NewPage":""
        }
    \#pdb.set_trace()
    yield scrapy.FormRequest(
        url = "http://jzzb.cqjsxx.com/CQ_ZB/ForeDisplay/ZBAffiche_Info/ZBAffiche_Info.aspx",
        formdata = postdata_refresh,
        callback = self.get_url,
        meta = {"page":page,"project":project,"Viewstate":Viewstate},
        )

def get_url(self,response):
    print("...................这里是get_url.........................")
    project = response.meta["project"]
    page = response.meta["page"]
    item = TaskThreeItem()
    Viewstate = ["0"]
    Viewstate[0] = response.xpath("//*[@id='form1']/input[3]/@value").extract()
    if project<3:

        url = response.xpath("/html/head/script/text()").extract()
        url = re.compile("http.*?(?=\')").findall(url[0])
        \#pdb.set_trace()
        yield Request(url = url[0],callback = self.get_data,dont_filter=True,meta={"page":page,"project":project,"Viewstate":Viewstate})
    else:
        project = 1
        page = page+1
        if page == 10:
            return
        postdata_next = {
        "__VIEWSTATEGENERATOR":"77E10603",
        "__EVENTARGUMENT":"",
        "__EVENTTARGET":"Pager1:LB_Next",
        '__VIEWSTATE':Viewstate[0],
        "FNam":"",
        "Pager1:NewPage":""
        }
        \#pdb.set_trace()
        yield scrapy.FormRequest(
        url = "http://jzzb.cqjsxx.com/CQ_ZB/ForeDisplay/ZBAffiche_Info/ZBAffiche_Info.aspx",
        formdata = postdata_next,
        callback = self.parse,

        formdata = postdata_refresh,
        callback = self.refresh,

        )
