比较运算符

= <=> <> != < <= > >=
左边的数和右边的数进行比较，比较结果为真返回1，为假返回0，其余情况放回null

等号运算符（=）

  • 若等号两边都是字符串，则按照ASCII码进行比较
  • 若等号两边既有数字又有字符串，则将字符串转换为数字进行比较，若转换不成功将其看做0
  • 若等号任意一边有null，则结果为null

    安全等与（<=>）

  • 与等号运算符相等，但是能跟null进行比较，两边都是null返回1，一边null返回0

    非符号类型运算符

  • ISNULL / IS NOT NULL / IS NULL
